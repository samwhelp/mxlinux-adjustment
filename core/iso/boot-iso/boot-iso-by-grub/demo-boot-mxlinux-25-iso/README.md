

# demo-boot-mxlinux-iso




## Live Account

| Account  | Value  |
| -------- | ------ |
| Username | `demo` |
| Password | `demo` |

run to change password

``` sh
sudo passwd $(whoami)
```


run to remove password

``` sh
sudo passwd -d $(whoami)
```
